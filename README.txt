1 - Importer le projet en eclipse comme maven -> Existing Maven Project
2 - Creer une base de données en MySQL ou MariaDB sous le nom GestionVols
3 - Changer le mot de passe dans le fichier de configuration 
	Chemin Fichier : Gestion Vols/src/main/resources/application.properties
	
	Ligne a changer : spring.datasource.password=youssef93

4 - Run Project -> Run as Spring boot project
5 - L'application va creer une base de données dynamiquement 
6 - il faut inserer au minimum un utilisateur agent dans la base manuellement pour pouvoir connecter
